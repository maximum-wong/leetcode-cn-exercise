#### 326. 3的幂(Power of Three)

给定一个整数，写一个函数来判断它是否是 3 的幂次方。<br/>
Given an integer, write a function to determine if it is a power of three.

**示例 1(Example 1):**

```
输入(Input): 27
输出(Output): true
```

**示例 2(Example 2):**

```
输入(Input): 0
输出(Output): false
```

**示例 3(Example 3):**

```
输入(Input): 9
输出(Output): true
```

**示例 4(Example 4):**

```
输入(Input): 45
输出(Output): false
```

你能不使用循环或者递归来完成本题吗？

**Follow up:**<br/>
