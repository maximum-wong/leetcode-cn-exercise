#### 342. 4的幂

给定一个整数 (32 位有符号整数)，请编写一个函数来判断它是否是 4 的幂次方。<br/>
Given an integer (signed 32 bits), write a function to check whether it is a power of 4.

**示例 1(Example 1):**

```
输入(Input): 16
输出(Output): true
```

**示例 2(Example 2):**

```
输入(Input): 5
输出(Output): false
```
